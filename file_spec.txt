file format specification (inspired by PNGs):

signature (8 Bytes) : ''

IHDR chunk

other chunks (including at least one IDAT)

IEND chunk


chunk format:
 - chunk size (4 Bytes unisgned integer) : size of chunk data *only*
 - chunk type (4 Bytes corresponding to ASCII uppercase and lowercase letters) : describes chunk type (e.g. IDAT, IHDR, ...)
 - chunk data ([chunk size] Bytes) : main content of the chunk (can be 0 lentgh if chunk size is 0)
 - CRC (4 Bytes) : CRC caclulated on preceding chunk bytes (excluding chunk size, but not chunk type); can be used to check for corrupted data
