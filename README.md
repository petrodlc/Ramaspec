A few python files that provide useful utilities to process Ramam spectrums

# Installation

## Install
clone from GitHub:

git clone https://github.com/petrodlc/Ramaspec.git

## dependencies
- numpy
- scipy
- pathlib

## update

git pull

## uninstall

remove directory

# Usage

import python files in your script, or from python console for live processing

import spectra as sp
